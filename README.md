# Murad Hasan Portfolio

[Live Demo](https://muradhasan49.netlify.app)

## Overview
This is the personal portfolio of **Murad Hasan**, showcasing projects, skills, and professional experience. Built with **HTML, CSS, JavaScript, and Bootstrap**, the site is fully responsive and optimized for modern web standards.

## Features
- Clean and modern UI design
- Fully responsive layout for mobile, tablet, and desktop
- Interactive navigation and sections
- Showcase of projects with live demo links
- Contact section with form integration
- Smooth scroll and hover effects

## Technologies Used
- **HTML5** – Structure and semantics
- **CSS3** – Styling and animations
- **JavaScript** – Interactivity and DOM manipulation
- **Bootstrap 5** – Responsive grid system, components, and utilities
